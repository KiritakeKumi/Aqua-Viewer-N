# Aqua-Viewer-N


Aqua Viewer是Aqua Server的网页UI 不过进行了一些改动和修正

这个项目基于 Angular.


## 想用？从源代码进行构建


你必须安装Angular-cli然后运行 `ng build`.
一般来说 安装后检查相关组件有没有装上就没啥问题了.
